# Juan Morales Portfolio Site

This is your personal portfolio website built with the **Modern Dark** theme.

## 📁 Files Included
- `index.html` — your main website file
- `StarterImage_general project thumbnail.png` — project thumbnail image

## 🌐 Publishing on GitHub Pages
1. Create a GitHub repository named `juan-morales-portfolio` (or any name you like).
2. Upload these files to the main branch.
3. Go to **Settings → Pages** in GitHub.
4. Under “Source,” choose **Deploy from a branch** → `main` → `/ (root)`.
5. Click **Save** — your site will be live in a few minutes at  
   `https://<your-username>.github.io/<repo-name>`.

## 🛠 Updating Your Site
To make changes, simply edit `index.html` and re-upload.
